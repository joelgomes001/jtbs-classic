## 📥 Install the Android App

Get the official *JTBS Classic* app directly on your Android device for the best viewing experience.

### 👉 Download JTBS Classic APK

https://github.com/joelgomes001/jtbs-classic/releases/latest/download/JTBS-Classic.apk

### How to Install on Android Phone / Tablet

1. Click the download link above.
2. Open the downloaded file (JTBS-Classic.apk).
3. If prompted, allow your browser to install apps from unknown sources.
4. Tap *Install* and launch the application.

### How to Install on Android TV / Smart Box

1. Download the APK file to a USB flash drive.
2. Connect the USB drive to your Android TV or Smart Box.
3. Open the APK using a file manager application and proceed with installation.
